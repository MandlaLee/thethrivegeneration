# Kingship Christian Centre — Bundle 10

Bundle 10 completes the first full website build with media, sermons, gallery, legal, SEO, favicon, domain, and launch-support files.

## Included pages

- `media.html`
- `gallery.html`
- `sermons.html`
- `privacy-policy.html`
- `terms.html`
- `404.html`

## Included systems

- Filterable media library
- Filterable sermon library
- Gallery with category filter and lightbox
- Privacy and terms placeholders
- SEO sitemap and robots file
- GitHub Pages `CNAME`
- Web manifest
- Favicons and placeholder media assets

## Important manual checks

1. Merge this bundle into the same project root as Bundles 1–9.
2. Replace all placeholder images and sermon/media entries.
3. Review the privacy policy and terms before collecting real user data.
4. Confirm that GitHub Pages is configured for the `main` branch.
5. Add the domain DNS records required by GitHub Pages.
6. Ensure every HTML page links the manifest and favicon files during final polish.
7. Confirm the final service time, venue, banking details, leadership details, and event dates before launch.
